Church Source is an open source church management web application.  The application is in heavy development right now and just focusing on a child check in/out system.  That is the current need at the church I attend, Dominion Church, so that is where that application started.  However, eventually I hope to add other features to give churches a free, open source alternative to expensive, propriety church management applications.

Future possible features:

  • Group and Person Management
  • Volunteer Management
  • Donation/Financial Tracking
  • Mailing List Management

Church Source is written using the Django framework and can be installed on your own server using standard technologies such as MySQL, Postgres, Apache, etc.

If you would like to get involved in the project see the following:

  • Git Hub: report issues, review source code
  • Wiki: Installation and other useful info
  • Google Moderator to suggest and vote on new ideas.
